# Memory Policy

## Allowed automatic learning

- Glossary terms explicitly defined in a cited source
- Non-sensitive user output preferences
- System names, ownership, and source links stated unambiguously
- Decisions explicitly marked approved

## Governed learning

Create a proposal for templates, policies, business rules, agent instructions, conflicting claims, confidence below `0.8`, or facts inferred from behavior.

## Exclusions

Never retain credentials, tokens, private keys, personal contact/payment data, chat pleasantries, duplicated boilerplate, unsupported conclusions, or content marked confidential unless an approved policy explicitly allows it.

Deprecate superseded facts. Keep provenance and audit history.
